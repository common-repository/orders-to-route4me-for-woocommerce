Send WooCommerce orders to Route4Me for instantaneous order to route integration.

== Description ==

Orders to Route4Me for WooCommerce helps you to send WooCommerce orders to Route4Me for instantaneous order to route integration. The plugin has built in geocoding functionality to make sure orders will pin on the map. When being exported, orders can be further edited or you can plan a route consisting of your orders and make a successful delivery.

= About Route4Me =
With the fastest dynamic route optimization and its intelligent tools [Route4Me](http://www.route4me.com/ "Route Planning Sofware") is the best route planner you can find, a sofware already proven by thousands of small business owners that are no longer suffering and wasting time planning routes. 

= Features =
1. Choose to automatically send newly added orders to Route4Me
2. Send order from a single order page screen
3. Send multiple orders from orders list screen
4. Built in geocoder
5. Instant order status lookup on Route4Me

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Route4Me screen to insert the API key
